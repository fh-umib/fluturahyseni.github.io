# Catch Flights — Travel Website

A responsive multi-page travel website designed to help users explore destinations, flights, hotels, tickets, favorites, and travel suggestions through a clear and engaging interface.

The project was developed as a frontend web application using HTML and CSS and is deployed publicly through GitHub Pages.

---

## Live Demo

[View the Live Website](https://fh-umib.github.io/fluturahyseni.github.io/)

---

## Project Overview

Catch Flights is a travel-focused website that provides users with different sections for exploring travel-related information and services.

The website includes multiple connected pages for destinations, flights, hotels, booking options, favorites, travel suggestions, and additional information.

The project focuses on:

- Multi-page website development
- Responsive frontend design
- Clear navigation
- Organized page structure
- Consistent visual styling
- User-friendly travel content

---

## Main Features

- Explore travel destinations
- Browse available flight information
- View hotel-related content
- Explore ticket and booking options
- Save or view favorite destinations
- Access travel suggestions
- Navigate through multiple connected pages
- Responsive layout for different screen sizes
- Consistent design across the website

## Design and Development

The website was developed using separate HTML and CSS files for the main sections.

This approach helped organize the project into individual pages while maintaining a consistent visual identity throughout the website.

The development process included:

- Creating page layouts
- Designing navigation elements
- Organizing travel-related content
- Working with images and visual elements
- Styling individual website sections
- Connecting multiple HTML pages
- Improving responsiveness
- Deploying the website using GitHub Pages
